## What This Is

Column definition and mapping library for tabular data operations. Provides column metadata, ordering, and type mapping used by export and data libraries.

## When to Use

- Defining column schemas for exports
- Column type mapping and metadata
- Used internally by `AMCode.Exports` and `AMCode.Data`

## Verification

```bash
cd repos/amcode-library
dotnet build columnslibrary/AMCode.Columns/AMCode.Columns.csproj
```
